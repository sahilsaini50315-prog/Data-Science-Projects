# Car Price Prediction using Machine Learning

## Objective
Predict the selling price of a car using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
car data.csv

## Machine Learning Model
Linear Regression

## Evaluation Metrics
- MAE
- MSE
- R² Score

## How to Run
python car_price_prediction.py
